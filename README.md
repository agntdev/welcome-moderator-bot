# Welcome Moderator Bot

Telegram group moderation bot: press-to-verify onboarding, blocks media until confirmation, detects spam/floods/links and applies owner-configured automated penalties.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
